# CLI To-Do List Manager

A simple, lightweight Command Line Interface (CLI) application built with Python to manage your daily tasks. This project tracks tasks using a dictionary and saves them to a local text file.

#Features
- **Add Tasks**: Quickly save new tasks to your list.
- **View Tasks**: See all your tasks with their unique IDs.
- **Remove Tasks**: Delete tasks using their specific ID.
- **Data Persistence**: Tasks are saved to `tasksFILE.txt`.

#How to Run
1. Make sure you have Python installed.
2. Clone this repository or download the `toDo.py` file.
3. Open your terminal/command prompt.
4. Run the script using:

           python toDo.py
